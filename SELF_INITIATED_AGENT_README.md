## 🏃‍♂️ Quick Start

### 1. Install Dependencies
```bash
# Install core requirements
pip install -r requirements-self-initiated.txt

# Or install individually
pip install streamlit plotly networkx openai arxiv wikipedia-api scholarly newspaper3k
```

### 2. Set Up API Key
```bash
# Option 1: Environment variable
export OPENAI_API_KEY="your-openai-api-key-here"

# Option 2: Streamlit secrets (for deployment)
# Create .streamlit/secrets.toml with:
# OPENAI_API_KEY = "your-openai-api-key-here"
```

### 3. Test the System
```bash
# Verify everything works
python test_self_initiated_app.py
```

### 4. Launch the Advanced Interface
```bash
# Start the self-initiated research agent
streamlit run self_initiated_streamlit_app.py
```

   - Next steps recommendations
